# Laporan Praktikum Sistem Terdistribusi Dan Terdesentralisasi

Repository ini berisi dokumentasi dan laporan praktikum untuk mata kuliah **Praktikum Sistem Terdistribusi Dan Terdesentralisasi**. Seluruh aktivitas praktikum selama satu semester dicatat dan disusun dalam repository ini menggunakan sistem version control Git sehingga setiap perkembangan dan perubahan dapat terdokumentasi dengan baik.

Setiap minggu praktikum disimpan dalam direktori terpisah yang diberi nama sesuai nomor minggu praktikum (01, 02, 03, dan seterusnya). Di dalam setiap direktori terdapat file **README.md** yang menjelaskan kegiatan praktikum yang dilakukan pada minggu tersebut, serta dapat dilengkapi dengan file tambahan seperti kode program, catatan, maupun screenshot hasil praktikum.

Struktur ini dibuat agar laporan praktikum tersusun secara sistematis, mudah dibaca, dan memudahkan proses evaluasi oleh dosen.
